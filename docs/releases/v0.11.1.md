# Centrifugal Pump Sizing App v0.11.1

Release date: 2026-07-11

## Release Summary

Version 0.11.1 delivers the v0.11 engineering-model upgrade together with a
stable responsive toolbar and three persistent interface themes. The offline
HTML package contains the complete application and can be opened directly in a
modern browser without installation or network access.

## Major Engineering Upgrades

- Separate installed, operating, and standby pump counts.
- Independent required duty and predicted operating point.
- SI/US catalog-curve conversion with percentage efficiency entry.
- Blocking engineering validation and explicit catalog-row errors.
- Independent NPSH ratio and absolute-margin results.
- Worst-case rated NPSH scenarios.
- Configurable POR, AOR, MCSF, thermal minimum, and suction-specific-speed limits.
- Maximum absorbed-power motor-envelope sizing.
- Minimum, normal, and maximum liquid-level scenarios.
- Parallel one-pump, one-unavailable, and all-running staging.
- Conservative parallel branch-imbalance loading.
- Per-branch or total-flow equipment-loss basis.
- Control-valve Kv/Cv and clean/dirty equipment pressure loss.
- Separate suction and discharge roughness.
- Fluid correlation-range, freezing, and boiling safeguards.
- Expanded printable engineering report and screening disclaimer.
- Updated offline standalone application.

## Interface and Workflow Improvements

- Engineering Paper, Control Room Dark, and Blueprint themes.
- Compact accessible theme selector with persistent preference.
- Stable fixed-width Print Report / PDF control.
- Case notifications moved to a dismissible toast outside toolbar layout.
- Responsive case controls with truncation and tooltips.
